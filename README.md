## 📌 About this Project

This portfolio presents a practical implementation of an Information Security Management System (ISMS) based on ISO/IEC 27001:2022.

The project was developed from a real vulnerability disclosure case in a Brazilian e-commerce operating in a production environment.

It demonstrates how real-world vulnerabilities can be analyzed, assessed, and mapped to ISO 27001 controls.
